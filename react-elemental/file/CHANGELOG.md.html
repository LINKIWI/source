      9 ## v1.2.2
     10 
     11 ### Added
     12 
     13 * Support ref forwarding to the underlying native DOM element for `Button`, `Checkbox`, `Image`, `Link`, `TextArea`, `TextField`
     14 * `durations` and `timing` as newly exported transition constants
     15 
     16 ### Changed
     17 
     18 * `Modal` no longer shows a close icon when non-persistent (this is now abstracted out to the client)
     19 * `SelectList` inherits parent width more naturally, and no longer requires an explicit numeric width be set
     20 * `SelectList` supports `inverted` prop, to expand the selection options upwards rather than downwards
     21 * `Tag` enables transitions, for consistency with other components
     22 * Standardize transitions across all components with custom easing function
     23 * Styles are now injected at bootstrap time, rather than at dependent component mount time
     24 
     25 ## v1.2.1
     26 
     27 ### Changed
     28 
     29 * `Image` hides the alt text after the image has loaded
     30 * `RadioGroup` button no longer becomes horizontally compressed for small widths
     31 * `RadioGroup` no longer invokes the change callback for options that are already selected on click
     32 
     33 ## v1.2.0
     34 
     35 ### Added
     36 
     37 * `Image` component for displaying external images
     38 * `RadioGroup` component for radio controls
     39 
     40 ### Changed
     41 
     42 * `Checkbox` is slightly smaller in size
     43 * `RadioGroup` button is slightly larger in size
     44 * Default `Button` type to `button`
     45 * Respect semantic `disabled` attribute on `Button` when disabled
     46 * `TextArea` defaults to the primary font style, instead of the secondary font style
     47 * Support a secondary `TextArea` style
     48 * Fixed secondary `TextField` missing `box-sizing` property
     49 * Better secondary `TextField` adaptation to dark backgrounds
     50 * Bumped `react` peer dependency version to `^16.4.0`
     51 
     52 ## v1.1.2
     53 
     54 ### Added
     55 
     56 * Secondary style on `Tabs`
     57 * New prop `fit` on `Tabs` to automatically size the widths of each tab item
     58 * `Tabs` primary style now has a subtle hover effect
     59 
     60 ### Changed
     61 
     62 * Fixed inconsistent border radius on `Tabs`
     63 * Fixed inability to override padding on `Button`
     64 * Secondary `TextField` now has a transparent background by default
     65 
     66 ## v1.1.1
     67 
     68 ### Added
     69 
     70 * New `Elemental` component for a declarative abstraction over bootstrapping
     71 
     72 ### Changed
     73 
     74 * `Button` no longer automatically blurs on `mouseleave` events
     75 * Unnecessary explicit `Button` prop `onClick` removed
     76 * `Button` applies click style for touch events
     77 * `Button` now has a dedicated focus style, identical to the existing hover style
     78 * Minor tweaks to the hover and active appearance of `Button`
     79 * Fixed regression introduced in `Modal` where it would not expand to full height when the viewport height is too small
     80 * `Checkbox` no longer uses a pointer hand when it is `disabled`
     81 * `Link` has dedicated focus and active styles when using keyboard for navigation
     82 * `Link` applies click style for touch events
     83 * `Link` transition speed increased slightly
     84 
     85 ### Removed
     86 
     87 * `react-elemental` no longer depends on `react-icons` and `radium`
     88 
     89 ## v1.1.0
     90 
     91 ### Added
     92 
     93 * New `Spinner` component, which is now actually a spinner
     94 
     95 ### Changed
     96 
     97 * `Spinner` is now renamed to `Pulsator`
     98 * `Pulsator` takes prop `inactive` instead of `pulsate`; `inactive={true}` now has the same semantics as `pulsate={false}`
     99 * `Toast` now adds inner spacing with `padding` rather than `margin`

    101 ## v1.0.1

    108 ## v1.0.0
    109 
    110 ### Added
    111 
    112 * `Label` component for text labels associated with input elements
    113 * `LoadingBar` accepts new props `duration` and `delay` to override the animation duration and bounce delay, respectively
    114 * `Tabs` component for segmenting option choices within the same semantic hierarchy
    115 
    116 ### Changed
    117 
    118 * `Alert` no longer automatically adds a bottom margin, and delegates the responsibility of visibility (in the case of dismissible alerts) to the parent component
    119 * `Button` secondary style is now sized identically to the primary style, and click outlines are removed
    120 * `Checkbox` is now a controlled component; imperative APIs are deprecated
    121 * `Link` now delegates the responsibility of styling to the parent component, and does not make any assumptions about font styling
    122 * Fixed bug with `SelectList` where occasionally an option would not be selected despite clicking on it
    123 * `SelectList` no longer fires `onChange` when clicking on the placeholder
    124 * `Tag` now delegates the responsibility of visibility (in the case of dismissible tags) to the parent component
    125 * Fixed nonexsitent hover/focus styles on `TextArea`
    126 
    127 ### Removed
    128 
    129 * `LoadingBar`'s `width` and `position` props are deprecated in favor of passing them in directly as keys in a style override object
    130 * `TextField`, `TextArea`, and `SelectList` no longer accept `label` and `sublabel`, in favor of the dedicated `Label` component
