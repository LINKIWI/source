      8 **React Elemental** is a modern, flat UI library built for React. It is built for maximum development velocity: there is no CSS to import and no additional parameters to add to your Webpack configuration. React Elemental has first-class support for ES6/JSX and works effortlessly with SSR.
      9 
     10 See the [official documentation site](https://react-elemental-docs.static.kevinlin.info) for demos, examples, and detailed documentation for each available UI component.
     11 
     12 ![React Elemental](https://static.kevinlin.info/blog/react-elemental/hero.png)

     14 ## Installation
     15 
     16 ```bash
     17 $ npm install --save react-elemental
     18 ```
     19 
     20 Note that the Elemental fonts are not bundled directly with the component library. This cuts down
     21 the size of `react-elemental` and allows you to inject fonts of your choosing into the library. To
     22 use the standard Elemental fonts, see the installation instructions of
     23 [`react-elemental-fonts`](https://www.npmjs.com/package/react-elemental-fonts).

     25 ## Usage
     26 
     27 React Elemental exports UI components that you can use directly in your `render` function. However, you must first bootstrap (initialize) the library in order to set configuration globals. You can do this either *declaratively* using the provided `Elemental` React component (recommended) or *imperatively* by invoking the exported `bootstrap` function.
     28 
     29 #### Using the `Elemental` component
     30 
     31 At the top level of your application, instantiate an `Elemental` component and pass the rest of your application as its children. Configuration parameters are specified as props (available options are defined below).
     32 
     33 ```javascript
     34 import React from 'react';
     35 import { Elemental } from 'react-elemental';
     36 
     37 const App = () => (
     38   <Elemental {...configOpts}>
     39     application children
     40   </Elemental>
     41 );
     42 
     43 export default App;
     44 ```
     45 
     46 #### Using `bootstrap` directly
     47 
     48 As early as possible in your client-side rendering path, invoke the bootstrapping function (available options are defined below).
     49 
     50 ```javascript
     51 import React from 'react';
     52 import { bootstrap } from 'react-elemental';
     53 
     54 bootstrap(...);
     55 
     56 const App = () => {
     57   ...
     58 };
     59 
     60 export default App;
     61 ```
     62 
     63 #### Using exported Elemental UI components
     64 
     65 ```javascript
     66 import React from 'react';
     67 import { Button, Spacing, Text } from 'react-elemental';
     68 
     69 const MyComponent = ({ onClick }) => (
     70   <Spacing bottom>
     71     <Text size="epsilon" bold>
     72       Hello world!
     73     </Text>
     74     <Button
     75       text="Click me"
     76       onClick={onClick}
     77     />
     78   </Spacing>
     79 );
     80 
     81 export default MyComponent;
     82 ```

     84 ## Options
     85 
     86 To optionally specify overrides for global constants, you can pass props to the `Elemental` top-level component, or invoke the bootstrapping function with parameters:
     87 
     88 ```javascript
     89 import React from 'react';
     90 import { bootstrap } from 'react-elemental';
     91 
     92 // Declarative
     93 <Elemental fontOpts={fontOpts} colorOpts={colorOpts}>
     94   children
     95 </Elemental>
     96 
     97 // Imperative
     98 bootstrap(fontOpts, colorOpts);
     99 ```
    100 
    101 All respected override options are as follows:
    102 
    103 #### `fontOpts`
    104 
    105 |Key|Value|Example|
    106 |-|-|-|
    107 |`primary.regular`|URL to the primary regular font.|`'url(data:application/x-font-ttf;base64,...)'`|
    108 |`primary.bold`|URL to the primary bold font.|`'url(data:application/x-font-ttf;base64,...)'`|
    109 |`secondary.regular`|URL to the secondary regular font.|`'url(data:application/x-font-ttf;base64,...)'`|
    110 |`secondary.regular`|URL to the secondary bold font.|`'url(data:application/x-font-ttf;base64,...)'`|
    111 
    112 #### `colorOpts`
    113 
    114 |Key|Value|Example|
    115 |-|-|-|
    116 |`primary`|Hex color code string for the desired primary color.|`'#3eb8f0'`|
    117 |`primaryLight`|Hex color code string for the desired primary light color.|`'#d6ecf5'`|
    118 |`primaryDark`|Hex color code string for the desired primary dark color.|`'#036996'`|

    120 ## Babel plugin
    121 
    122 You may **optionally** include [`babel-plugin-react-elemental`](https://github.com/LINKIWI/babel-plugin-react-elemental) with your project to automatically transform imports so that only the components used by your application are bundled. This reduces the final application bundle size by reducing the library footprint.

    124 ## Development
    125 
    126 It is easiest to symbolically link the sample project's `react-elemental` dependency to the root project. Then, auto-transpile on source changes and auto-rebuild the sample project on dependency changes.
    127 
    128 ```bash
    129 $ npm install
    130 $ npm run build:watch &
    131 $ cd sample
    132 $ npm install
    133 $ rm -rf node_modules/react-elemental
    134 $ ln -s ../.. node_modules/react-elemental
    135 $ npm run start
    136 ```
